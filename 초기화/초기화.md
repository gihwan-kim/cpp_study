# 1. uniform initialization

1. {} 를 사용하여 초기화를 하는 방식

2. 구조체, 클래스, 다른 대상을 초기화할때 사용할 수 있다.

3. 등호(=) 를 사용하거나 사용하지 않아도 된다.

```
int a = {1};
int a{1};
```

4. 축소 변환을 방지해준다.
> 큰 데이터형의 상수가 작은 데이터형의 변수에 담기지 않도록 컴파일러에서 에러 검출
```
int a = {3.3}; // 에러
```


5. 배열을 초기화할 때도 사용
```
int array[3]{1,2,3};
```

### copy list initialization
```
T obj = {arg1, arg2, ...};
```
- 중괄호 안의 원소들의 타입이 모두 같아야한다.


### direct list initialization
```
T obj{arg1, arg2, ...};
```
